WHO WOULD USE THIS DEVICE?

This project would be a pedometer device (step counter) that will count the users steps and store their total steps and we will add achievements and/or goals to be met along with an image and music that will play once the goal each step counting goal is met.

HOW DOES THIS PROJECT WORK?

Our pedometer that we plan to create would have various sets of goals when certain amounts of steps are taken. The project will work by using the shake input to detect the steps that were taken and return back various feedback when the number of shakes/steps are reached then our output would be music that would play along with the image on the LED for whenever the step goal is reached.
